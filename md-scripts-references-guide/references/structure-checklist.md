# Structure checklist

Use this checklist when creating or updating a skill:

- Skill folder name uses lowercase letters, digits, and hyphens.
- `SKILL.md` exists at the skill root.
- Frontmatter includes only `name` and `description`.
- `scripts/` exists only when executable helpers are needed.
- `references/` exists only when long-form docs are needed.
- `SKILL.md` links to each reference file directly.
- No extra process docs (README/changelog/install notes) unless explicitly requested.
